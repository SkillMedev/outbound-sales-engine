# Outbound Sales Engine

**For AEs and SDRs: run a B2B outbound deal from cold email to close with sharper execution.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

Reach for this when you own outbound deals and want every stage to land — not generic sales theory. It walks the full motion: write cold outreach that gets replies, prep discovery so you qualify and tailor, demo to the buyer's actual pain, defuse the price/timing/competitor/status-quo objections that stall deals, write a proposal that drives a next step, and run a mutual action plan plus a value-adding follow-up cadence that gets the deal signed instead of going quiet. Built for AEs and SDRs who want sharper execution under deal pressure.

## Install

- **From the catalog:** [skillme.dev/pack/outbound-sales-engine](https://skillme.dev/pack/outbound-sales-engine) — install the whole pack into Claude in one step.

## Skills in this pack

- **[Discovery Call Prep](skills/discovery-call-prep/SKILL.md)** — Prepares a rep for a B2B discovery call — company research, hypothesis stack, question plan, and clear call goals.
- **[Objection Handler](skills/objection-handler/SKILL.md)** — Handles common B2B sales objections — price, timing, incumbent competitor, and status quo — with calibrated, non-pushy responses.
- **[Sales Proposal Writer](skills/sales-proposal-writer/SKILL.md)** — Writes a crisp B2B proposal or SOW that leads with the buyer's problem, frames value clearly, defines scope, and drives a specific next step.
- **[Mutual Action Plan](skills/mutual-action-plan/SKILL.md)** — Builds a mutual action plan (MAP) with shared milestones, owners, and dates to drive a deal to close.
- **[Demo Script](skills/demo-script/SKILL.md)** — Structures a product demo around the buyer's specific use case and pain, not a feature tour.
- **[Sales Follow-Up Cadence](skills/sales-followup-cadence/SKILL.md)** — Designs a multi-touch follow-up cadence that adds value at each touch, maintains momentum after a call or demo, and knows when to stop.
- **[Cold Email Craft](skills/cold-email-craft/SKILL.md)** — B2B cold outreach that gets replies: tight personalization, brevity, and a low-friction ask.
